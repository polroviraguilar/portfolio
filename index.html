Monkey Island Fight

About

Monkey Island Fight is a solo project that reimagines the classic insult-sword-fighting mechanic from the legendary Monkey Island series. Developed during my Master’s program, this project is both a tribute and a technical exercise in narrative design.

Project Info

👤 Role: Solo Developer

👥 Team Size: 1

⏱️ Time Frame: 2 Months

⚙️ Engine: Unity (C#)

Introduction

My goal was to faithfully recreate the charm and rhythm of the original insult fights. The game had to feel reactive, funny, and sharp — like a dance of words. I wanted the player to feel clever, not just win.

Dialogue System

I designed a custom branching dialogue system that allows each insult and response to carry consequences, creating layered insult chains that escalate dynamically. The structure was built using C# scriptable objects and a simple state manager.

The logic structure behind the insult-response dialogue system.

User Interface & Feedback

The UI was kept clean and retro-styled to reflect the original’s aesthetic. I added subtle animations and sound cues to reinforce the rhythm of exchange. Each insult and comeback has synced visual feedback to enhance punchlines.

Custom UI mockup showing dialogue options and timing feedback.

Timing & Delivery System

Timing matters in comedy. I built a delay-and-pause system to control delivery pacing — allowing jokes to land properly. Pauses between lines are dynamically calculated based on the player’s reading speed and choice timing.

Prototype showing dynamic timing between insults and comebacks.

What I Learned

This project helped me understand how humor is not just written — it’s engineered. Timing, UI pacing, player agency and visual feedback all matter in delivering something that feels funny and interactive. It also deepened my skills with state machines, dialogue trees, and Unity UI tooling.
