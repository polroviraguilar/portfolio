About

"Monkey Island Fight" was a solo game project built during my Master's degree. It reimagines the classic insult-sword-fighting mechanic from the Monkey Island series using Unity and C#.

You duel opponents not with weapons, but with wit â€” selecting the right comeback is key to victory.

Introduction

The goal was to faithfully recreate the spirit of Monkey Islandâ€™s legendary duels, where timing and insults replace traditional combat. I wanted to explore branching dialogue systems and inject humor into gameplay mechanics.

Dialogue System

The core challenge was building a flexible dialogue tree that tracks player choices, comeback logic, and opponent responses. This was built entirely in Unity with a custom C# backend and state machine.

UI & Animation

I created custom UI panels to display insult and response options. Character animations were timed to match dialogue beats for comedic effect, enhancing the parody.

What I Learned

This project sharpened my skills in game logic, UI/UX design, and humor-driven narrative systems. I gained a deep appreciation for how simple mechanics like branching dialogue can feel compelling when wrapped in strong writing.
